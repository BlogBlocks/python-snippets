String
======

Snippets on string manipulation.


`check if a string is a number`_
--------------------------------

::
    
    >>> a = "123"
    >>> a.isdigit()
    True
    >>> b = "123abc"
    >>> b.isdigit()
    False


`reverse a string`_
-------------------

::
    
    >>> 'hello world'[::-1]
    'dlrow olleh'
